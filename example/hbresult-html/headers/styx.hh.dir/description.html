styx.hh


Abstract: Top level header file of Styx library for XML manipulation (SAX, DOM and XPath).

Include this file to access all functionnalities of the Styx lib. Refere to styx_sax.hh and styx_dom.hh to know more about SAX parser, DOM tree construction, and XPath uses.
